Simple-clock(for Japanese)

#作者 指差喚呼Y

#特徴
都度コードを書かなくても最低限・シンプルな時計ができる、を実現しました。

#使い方
htmlファイルに、
```
<div id="y">
  <span id="hi"></span>
  <span id="clock"></span>
</div>
```
を追加してください。
そして、
```
<script type="module" src="https://shisakankoy.github.io/Simple-clock/main.js"></script>
```
も追加してください。
