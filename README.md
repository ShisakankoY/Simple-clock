#特徴
都度コードを書かなくても最低限・シンプルな時計ができる、を実現しました。

#使い方
htmlファイルに、
<div class="wrapper">
  <span id="hi" class="x"></span>
  <span id="clock" class="x"></span>
</div>
を追加し、また、style.cssを指定してください。
そして、
<script type="module" src="https://shisakankoy.github.io/Simple-clock/main.js"></script>
も追加してください。
